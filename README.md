# QuickReply 💬

A modern frontend productivity tool that helps users reply to emails faster using pre-built templates and dynamic reply generation.

## 🚀 Live Demo
https://janffy1.github.io/quickreply-tech/

## 🛠 Tech Stack
- HTML5
- CSS3 (Glassmorphism + Gradient UI)
- JavaScript (DOM Manipulation)
- GitHub Pages (Deployment)

## ✨ Features
- One-click copy email replies
- Dynamic reply generator
- Modern tech-style UI
- Fully deployed using GitHub Pages

## 📦 Deployment
Hosted via GitHub Pages from the main branch.
